## LMS

LMS (Learning management system) — система управления обучением, предназначенная для организации курсов, лекций и тестирования студентов.

## Оценка времени на задания 

Оценка времени на задания будет производиться по методу оценки по трём точкам. Для этого используется формула:

$$
E = (P + O + 4 * BG) /  6,
$$

где P — продолжительность работы над задачей, если сработают все выявленные негативные факторы;<br>
O — оптимистичная оценка, в которой учтены все определённые ранее позитивные факторы;<br>
BG — это наиболее вероятная, которая является средней продолжительностью работы над задачей, если работник сделает её 100 раз.<br>

Результаты представлены в таблице:

| БД          | - | - | -  | -                       | -                                |
| JDBC        | - | - | -  | -                       | -                                |
| Backend     | - | - | -  | -                       | -                                |
